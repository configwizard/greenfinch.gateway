Available Endpoints

Here are some examples of how to use the available endpoints:

Example OID: 7hhFNxefLDwHeFkU4FwA6H7FVKYWKefbq8kzKGyPybsD

  • Get container and object ID: /[CID]/[OID]
  • Get index file in a container: /[CID]
  • Get specific file from a container: /[CID]/image.jpg
  • Get file by Filepath: /[CID]/assets/styles.css
  • Specify attribute for a container: /attribute/[CID]/FileName/sami2.jpg
